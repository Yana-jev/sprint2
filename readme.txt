## Bloque 1.1: Funciones de flecha

### Nivel 1: Ejercicio 1
La función `add` toma dos parámetros y devuelve su suma. Conviértela en una función de flecha.

### Nivel 1: Ejercicio 2
La función `randomNumber` no toma parámetros y devuelve un número aleatorio entre 0 y 100.

### Nivel 1: Ejercicio 3
Crea la clase `Person` con un campo `name` y el método `greet`, que utiliza una función de flecha para imprimir un saludo con el nombre.

### Nivel 2: Ejercicio 4
La función `printNumbers` toma una matriz de números y utiliza un bucle `for` y una función de flecha para imprimir cada número.

### Nivel 3: Ejercicio 5
La función `sayHello` utiliza `setTimeout` y una función de flecha para imprimir el mensaje "say Hello" después de esperar 3 segundos.

## Bloque 1.2: Operador ternario

### Nivel 1: Ejercicio 1
La función `puedeConducir` acepta la edad como parámetro y utiliza el operador ternario para determinar si el usuario puede conducir.

### Nivel 1: Ejercicio 2
Escribe una expresión que utilice el operador ternario para determinar cuál de los dos números dados es mayor.

### Nivel 2: Ejercicio 3
Escribe una expresión que utilice enlaces de operadores ternarios para determinar si un número es positivo, negativo o cero.

### Nivel 2: Ejercicio 4
La función `encontrarMaximo` acepta tres parámetros y utiliza el operador ternario para determinar el valor máximo.

### Nivel 3: Ejercicio 4
La función `parOImpar` acepta un array de números y utiliza un bucle para determinar si cada número es par o impar.

## Bloque 1.3: Callbacks

### Nivel 1: Ejercicio 1
La función `procesar` acepta un número y una función de callback, y llama a la función de callback con el número como parámetro.

### Nivel 1: Ejercicio 2
La función `calculadora` acepta dos números y una función de callback, y llama a la función de callback con los dos números como parámetros.

### Nivel 2: Ejercicio 3
La función `esperarISaludar` acepta un nombre y una función de callback, y espera 2 segundos antes de llamar a la función de callback con el nombre como parámetro.

### Nivel 2: Ejercicio 4
La función `procesarElementos` acepta un array y una función de callback, y llama a la función de callback por cada elemento del array.

### Nivel 3: Ejercicio 5
La función `procesarCadena` acepta una cadena y una función de callback, convierte la cadena a mayúsculas y luego llama a la función de callback con la cadena transformada.

## Bloque 1.4: Operadores Rest & Spread

### Nivel 1: Ejercicio 1
Crea una nueva array que contenga todos los elementos de dos arrays utilizando el operador spread.

### Nivel 1: Ejercicio 2
La función `suma` acepta un número indeterminado de argumentos y devuelve su suma.

### Nivel 2: Ejercicio 3
Crea un nuevo objeto que sea una copia de otro objeto utilizando el operador spread.

### Nivel 2: Ejercicio 4
Utiliza el operador resto en una array para asignar los primeros dos elementos a variables y el resto a otra variable.

### Nivel 3: Ejercicio 5
Crea una función que acepte tres argumentos y llama a la función utilizando el operador spread con una array.

### Nivel 3: Ejercicio 6
Crea un nuevo objeto fusionando dos objetos existentes utilizando el operador spread.

## Bloque 1.5: Transformaciones de arrays

### Nivel 1: Ejercicio 1
Crea una nueva array que contenga el cuadrado de cada número de otra array utilizando `map`.

### Nivel 1: Ejercicio 2
Crea una nueva array que contenga solo los números pares de una array existente utilizando `filter`.

### Nivel 1: Ejercicio 3
Utiliza `find` para encontrar el primer número mayor que 10 en una array.

### Nivel 1: Ejercicio 4
Calcula la suma total de los números en una array utilizando `reduce`.

### Nivel 2: Ejercicio 5
Crea una función que filtre los números mayores o iguales a 10 en una array, los multiplique por 2 y devuelva la suma de estos números.
